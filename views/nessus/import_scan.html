Nessus scan files with the extension .nessus are best used for importing. When exporting from the Nessus Standalone GUI always select the Nessus format if Kvasir is not configured or able to talk directly to the console.

If CSV is the only format available Kvasir will support it. Should this export come from the Standalone scanner the data will be very limited. CSV output from the centralized management console is much better.
